Text Encoding, ASCII, Unicode

Example of inconsistent meta tag and http header

Example text with Scandinavian characters and Euro:

Some text זרו₪

The page is encoded (i.e. saved) in ISO-8859-15

It correctly contains the
<meta charset="iso-8859-15"/>
meta tag

but the HTTP server sends the HTTP header
content-type text/html; charset=utf-8
which overrides the meta tag set in the file.
